# 동성로 관광 안내 사이트

### 🔗링크
http://dongseong.dothome.co.kr/

### 🔎소개
대구 광역시의 동성로를 소개하는 사이트.

### 📅제작 기간
2024.03.04 ~ 2024.03.28

### 🗂개발 환경
- HTML
- CSS
- JavaScript
- PHP
- Mysql

### 🎈주요 기능
- 네이버, 카카오 연동 소셜 로그인
- 회원 전용 장소 찜목록 관리

### 💾활용 요소
- 네이버 & 카카오 로그인
- 대구광역시 맛집 API
- 대구광역시 전시정보 API
- Bootstrap
- Google Cloud API
- Swiper JS
